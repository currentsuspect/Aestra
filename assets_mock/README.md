# assets_mock

This folder contains small placeholder assets used when AestraPremium (models / large assets) is not present.

Purpose:
- Allow developers to build and run `Aestra-core` without private model files.
- Keep small, permissive placeholder assets (short WAV samples, tiny fonts, placeholder icons).

Usage:
- Leave this folder committed in the public repo. Private assets should be stored in the `AestraAssets` or private `Aestra-premium` repo.
